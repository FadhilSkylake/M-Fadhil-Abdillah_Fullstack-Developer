## Tentang Aplikasi ini

-   silahkan lakukan git pull terlebih dahulu
-   lalu export database yang bernama hospital.sql ke MySql
-   lalu buka folder project dan jalankan perintah composer install untuk instalasi

-   untuk menjalankan aplikasi nya php artisan serve
-   jalankan database
-   lalu buka http://127.0.0.1:8000/
-   untuk login bisa ke /login dengan mengisikan email fadhil1023@gmail.com dan password admin
